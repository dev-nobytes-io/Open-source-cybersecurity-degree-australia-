## How to Share This Profile

### As a GitHub Portfolio
1. Fork this repository (or create your own)
2. Complete this markdown file
3. Open `student-portfolio/index.html` for the visual/interactive version
4. Host on GitHub Pages for a public URL you can share with employers

### As a PDF
Export this markdown file to PDF using:
- VS Code with Markdown PDF extension
- Pandoc: `pandoc competency-profile.md -o competency-profile.pdf`
- Any markdown-to-PDF tool

### As a LinkedIn Addition
Add your top NICE DCWF work roles to your LinkedIn certifications/skills section,
linking to your GitHub portfolio URL as the issuing organisation's credential URL.
