## Assessment Approach

This major is less lab-intensive than operational majors. Assessments should focus on:

- Executive communication outputs (written reports, oral presentations, board briefings)
- Case study analysis (real or fictional incidents requiring CISO-level decision-making)
- Simulation exercises (board presentations, media handling, regulator notification)
- Peer review exercises (critiquing security strategies and investment cases)

Minimum requirements still apply: at least 1 formative + 1 summative assessment per unit with a 4-level rubric.

## Activity Design (In Lieu of Technical Labs)

This major requires a minimum of 8 practical activities across 6 units. These need not be technical labs — they can be simulation exercises, case studies, and structured deliverables:

| Unit | Activity Ideas |
|---|---|
| LD01 | Activity: Map the CISO reporting structure of 3 real Australian ASX-listed companies using public disclosures; identify structural differences |
| LD01 | Activity: Draft a CISO mandate document for a fictional Australian bank |
| LD02 | Activity: Build a 3-year security roadmap for a fictional organisation using a maturity model (CMMI or NIST CSF) |
| LD02 | Activity: Write a business case for a security investment using risk-adjusted language |
| LD03 | Activity: Convert a technical incident report into a 1-page board-level risk brief |
| LD03 | Activity: Design a CISO dashboard for a fictional organisation — choose metrics that matter to the board |
| LD04 | Activity: Design a security team structure for a given organisation size and risk profile |
| LD04 | Activity: Write a talent retention strategy for a fictional SOC team facing high attrition |
| LD05 | Activity: Simulate a breach notification scenario — write the OAIC notification, ASX disclosure, and media statement |
| LD05 | Activity: Participate in a tabletop crisis communication exercise |
| LD06 | Capstone: Present a complete security strategy, roadmap, and budget to a simulated board panel |

- [ ] **LD04** — Australian cybersecurity talent market; ASD workforce initiatives

## Capstone Unit (LD06) Requirements

The capstone is a **CISO simulation exercise**:

- [ ] Fictional (or anonymised) Australian organisation with defined sector and risk context
- [ ] Security strategy document produced (executive-ready)
  - Current state assessment (maturity model)
  - 3-year roadmap
  - Risk-prioritised investment plan with budget
- [ ] Board-level presentation delivered (minimum 15 minutes)
- [ ] Q&A with simulated board members (community practitioners playing board roles)
- [ ] Post-presentation reflection report written
- [ ] Deliverable suitable for professional portfolio (reference `templates/student-portfolio/index.html`)
- [ ] Community practitioner review (ideally a current or former CISO)
