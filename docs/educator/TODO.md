## assessment-moderation-guide.md

### Purpose

Ensures assessment marking is consistent across different community reviewers using the same rubrics.

### Actions

- [ ] **Define the moderation process** — how multiple reviewers calibrate their rubric interpretation before reviewing a cohort's submissions
- [ ] **Add an anchor submission library** — example submissions at each rubric level (Exemplary, Proficient, Developing, Beginning) for each assessment type

- [ ] **Add guidance on marking academic integrity issues** — what to do if a reviewer suspects plagiarism or AI-generated work submission
