# External Benchmarking

This document benchmarks the Open-Source Cybersecurity Degree against comparable
Australian and international cybersecurity programs. External benchmarking is a
requirement for TEQSA accreditation (HESF Domain 5) and for formal credit
recognition discussions with other institutions.

---

## 1. Purpose

External benchmarking serves three purposes:
1. Demonstrates that the degree meets or exceeds the academic standards of
   comparable AQF Level 7 programs
2. Identifies areas where the degree offers distinct value not present in comparators
3. Informs the ongoing curriculum development agenda

This document should be reviewed and updated annually, or when a comparable
institution makes significant curriculum changes.

---

## 2. Australian University Comparators

### UNSW Sydney — Bachelor of Cyber Security

| Dimension | UNSW | This Degree |
|---|---|---|
| Duration | 3 years full-time | Self-paced (~3 years) |
| Credit points | 144 CP (AQF 7) | 168 CP (AQF 7 equivalent) |
| Delivery | On-campus + online | Online / self-paced |
| Cost | ~$32,000–$40,000 AUD (domestic) | Free |
| Framework mapping | Implicit | Explicit (NICE DCWF T-code level) |
| Australian regulatory content | Included | Mandatory in every unit |
| Practitioner review | Not documented publicly | Mandatory (governance model) |
| MITRE ATT&CK integration | Selected units | Across all operational units |
| Student competency portfolio | Not standard | Core design requirement |
| Industry certifications bridge | Some | Explicit per major |

**Assessment:** UNSW's program has strong academic rigour and institutional
recognition. This degree offers comparable content depth with stronger framework
mapping, more explicit practitioner review, and greater flexibility and accessibility.

## 3. International Comparators

### CS50x (Harvard/edX) — Computer Science Introduction

This degree uses CS50 as its design reference. Key comparisons:

| Dimension | CS50x | This Degree |
|---|---|---|
| Cost | Free | Free |
| Academic rigour | High (Harvard-level) | AQF Level 7 aligned |
| Framework mapping | Not applicable (CS scope) | Explicit (NICE, SFIA, ASD) |
| Certification bridges | None formal | Explicit per major |
| Professional portfolio | Projects | Competency profile + portfolio |
| Formal recognition | Certificate of completion | None yet (pathway exists) |

**Assessment:** CS50 demonstrates that a free, rigorous, community-supported
curriculum can achieve global respect. This degree applies the same model to
a specialised, framework-mapped domain.

### UK Cyber Security Body of Knowledge (CyBOK)

CyBOK is not a degree but a structured knowledge framework used by UK universities.
Comparison is at the knowledge coverage level.

| CyBOK Knowledge Area | Coverage in This Degree |
|---|---|
| Human, Organisational & Regulatory Aspects | F05, GRC major, LD major |
| Attack & Defence | OC01–OC06, CE major, DE major |
| Security Operations & Incident Management | OC02, OC04, DF major, TH major |
| Software & Platform Security | SE major, F02, F03 |
| Infrastructure Security | F01, F02, SE major |
| Applied Cryptography | SE02, F04 (introductory) |
| Network Security | F01, OC02, DE major |
| Adversarial Behaviours | OC01, CT major, TH major |
| Security Management & Governance | SC01–SC06, GRC major, LD major |

**Assessment:** This degree covers the majority of CyBOK knowledge areas, with
strongest alignment in Operations, Attack & Defence, and Governance. Applied
Cryptography is covered at introductory level only — a gap acknowledged and
appropriate for a practitioner rather than research degree.

## 4. Distinctive Strengths of This Degree

Across all comparators, this degree has the following distinctive strengths:

| Strength | How it differs |
|---|---|
| **T-code level NICE DCWF mapping** | No Australian degree documents at this granularity |
| **ASD Cyber Skills Framework mandatory** | Most degrees do not reference this Australian-specific framework |
| **Practitioner review governance** | A formal, documented review process is unique among free programs |
| **Student competency portfolio system** | Structured NICE-to-evidence portfolio is not standard in formal degrees |
| **8 majors** | More specialisation options than most comparable Australian degrees |
| **Free** | Removes the primary access barrier for career changers and regional learners |
| **Fully open-source** | Enables university adoption, adaptation, and contribution |
